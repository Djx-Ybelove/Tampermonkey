# 百度文库下载脚本

## 使用方法

+ 首先要打开一个百度文档(word，pdf，ppt都可)。  
+ 右下角将会跳出下载文档的按钮。
![alt text](1710489528962.png)
+ 点击即可进入下载界面，点击获取文档，会进入身份验证界面。
![alt text](1710489720554.png)
+ 将口令复制进去即可完成验证。
+ 还提供pdf转换网站，点击即可进入。

****

## 注意事项

+ 此脚本下载文档次数有限(一天大概可以下十几次)，请勿浪费资源，不能使用爬虫等工具进行攻击。
